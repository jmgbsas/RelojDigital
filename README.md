"# RelojDigital y Alarma Digital" 

La alarma digital es ademas de un reloj digital, un reloj con alarma
Emite un Beep en la PC despues de un intervalo dado.
Para configurar clickear la hora y pregunta dos cosas los minutos del
intervalo de la alarma Repetitiva y el mensaje.
Si el intervalo es 0 funiona solo como Reloj borra la alarma.
Conviene para hacer intervalos de descanso en un trabajo en PC.
El mensaje puede ser el que se  desee  como recordatoriom el mensaje 
por Omision es " DESCANSA UN POCO VAMOS.."
